# Description

A Pocketmine-MP plugin that allows adds world protection to the server. Including block interaction and PvP.

# Commands/Permission

```
permissions:

  protectionplus.protect:

    description: Allows players to enable/disable general protection.

    default: op

  protectionplus.interaction:

    description: Allows players to enable/disable general interaction.

    default: op

  protectionplus.damage:

    description: Allows players to enable/disable general pvp.

    default: op

```
